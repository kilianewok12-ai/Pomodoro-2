Auf den Homescreen hinzufügen

So sparst du dir jedes Mal das Suchen

📱 iPhone / iPad (Safari)
  1. Öffne Safari und gehe auf die Twin Towers Hub Seite.
  2. Tippe auf das Teilen-Symbol (Quadrat mit Pfeil nach oben) unten in der Mitte.
  3. Wähle „Zum Home-Bildschirm".
  4. Tippe auf „Hinzufügen" – fertig! 🎉
🤖 Android (Chrome)
  1. Öffne Chrome und gehe auf die Seite.
  2. Tippe auf die drei Punkte oben rechts.
  3. Wähle „Zum Startbildschirm hinzufügen".
  4. Tippe auf „Hinzufügen" – fertig! 🎉
